Tempermetro
View Live View GitHub
Anyone who's ever taken the metro in Washington D.C. probably knows how frustrating and inconvenient it can be. Thus, I've created an app for people to vent their DC metro frustrations. This website was built in Rails in about a week. I used mediaqueries to make the site responsive.
When users come to this site they can log in and post a complaint about the metro. When a user posts a complaint about the DC metro system, he or she needs to select the color of the metro-line that that specific event occurred on. The color that he/she picks will become the border color of that post. This way, people who visit the site can easily tell which metro line has the most issues because there will be more posts with that metro-line color. Users can also comment on other people's post and delete or edit their own posts.

Rails
JavaScript
HTML5

CheatSS
View Live View GitHub
I've always wanted to create a site where I can post CSS tricks that I've learned while styling pages so I created CheatSS. CheatSS is a website created in Node.js using the Express framework and MongoDB database.
When users log in to CheatSS they have the option of creating a CSS tutorial or viewing tutorials that have already been created. When creating a tutorial, the user must enter in the HTML, CSS and JavaScript required to create the effect stated in the tutorial. The HTML, CSS and JavaScript code is then formatted to look like code in a text editor after it is submitted. I hope that this tutorial will one day be a valuable resource for people trying to learn CSS tricks.

About Maureen

Hello, my name is Maureen and I'm a front-end web developer and designer. I started making websites when I worked as a graphic designer and needed a place to show off my work to the world -- And there's no better place to do this than the world wide web!
Being the perfectionist artist that I am, I was never satisfied with the overall look of my portfolio website which prompted me to start over multiple times. Each time I started over, I had a whole new idea in mind which required hours of HTML, CSS and JavaScript research. As a result, I learned so many cool techniques that keep growing every time I start a new project.
Besides making websites, I also have a diverse set of skills that include design (in Photoshop and Illustrator), painting, and crafts (I make dream catchers and sling shots). I've also lived in Japan for seventeen years and Taiwan for two so I speak both Japanese and Chinese (Mandarin).
